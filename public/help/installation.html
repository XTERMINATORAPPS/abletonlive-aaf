Installation Help

Windows: SmartScreen Warning

Why do I see this warning?
Windows displays this warning for new software that hasn't been "verified" through an expensive code signing certificate. The app is completely safe.

When you first run AbletonLiveAAF, you may see:

"Windows protected your PC"

1 Click "More info"
2 Click "Run anyway"

This is a one-time step. Windows will remember your choice.

macOS: Gatekeeper Warning

Why do I see this warning?
macOS blocks apps from "unidentified developers" by default. This is normal for indie software.

If macOS says the app "can't be opened":

1 Right-click (or Control-click) on the app icon
2 Select "Open" from the menu
3 Click "Open" in the confirmation dialog

This is a one-time step. macOS will remember your choice.
